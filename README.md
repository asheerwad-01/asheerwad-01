## Getting Started

### Prerequisites

- Node.js v18+
- npm

### Installation & Local Development

```bash
# Clone the repository
git clone https://github.com/asheerwad-01/asheerwadportfolio2026.git

cd asheerwadportfolio2026

npm install

npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Build & Preview

```bash
# Production build
npm run build

# Preview production build locally
npm run preview
```
